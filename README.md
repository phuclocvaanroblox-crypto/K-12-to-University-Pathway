# K-12 to University Pathway

Dự án này là lộ trình tự học và phát triển hệ thống hỗ trợ giáo dục từ bậc phổ thông lên đại học, bao gồm nhiều ngôn ngữ lập trình và các kỹ thuật bảo mật hiện đại.

## Công nghệ sử dụng
- **Backend:** Go, Python, PHP
- **Frontend:** HTML, CSS, JavaScript
- **Bảo mật:** Triển khai băm mật khẩu với thuật toán **SHA-3 512** và **Salt** ngẫu nhiên để đảm bảo an toàn thông tin.

## Cấu trúc dự án
Dự án được tổ chức theo các thư mục chuyên biệt:
- `/Go`: Xử lý logic hệ thống tốc độ cao.
- `/Python`: Phân tích dữ liệu và các module bảo mật.
- `/PHP`: Quản lý API và cấu trúc database.
- `/Html, css, js`: Giao diện người dùng thân thiện.

## Bảo mật
Dự án ưu tiên tính bảo mật. Các file cấu hình nhạy cảm được tách biệt và các thông tin định danh người dùng được mã hóa bằng thuật toán SHA-3 512.

## Ghi chú
Dự án này là sản phẩm của quá trình tự học và liên tục cải tiến. Mọi góp ý để tối ưu hóa code đều rất được trân trọng.
